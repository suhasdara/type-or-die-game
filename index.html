In this game, you shall type words as fast as you can. You will be given a specific amount of time to type each word depending on the speed you choose in the starting prompt.

For every correctly typed word, you will get 1 point and for every word missed, you will lose 0.25 points.

Shortcuts
Open Page Starting a new game: Ctrl + Enter
Game Page Pausing game: Ctrl + .
Resuming game: Ctrl + ;
Increasing game speed: Ctrl + Alt + K
Decreasing game speed: Ctrl + Alt + M
